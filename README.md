# Dona Rose AI/ML Portfolio

This is a free personal portfolio website for GitHub Pages.

## Files included

- `index.html`
- `style.css`
- `assets/favicon.svg`

## What to replace before publishing

Search in `index.html` and replace:

- `YOUR_RESUME_LINK_HERE`
- `YOUR_LINKEDIN_LINK_HERE`
- `YOUR_EMAIL_HERE`
- `YOUR_LEGAL_RAG_REPO_LINK`
- `YOUR_LEGAL_RAG_DEMO_LINK`
- `YOUR_PPE_REPO_LINK`
- `YOUR_PPE_DEMO_LINK`
- `YOUR_PREDICTIVE_MAINTENANCE_REPO_LINK`
- `YOUR_PREDICTIVE_MAINTENANCE_DEMO_LINK`
- `YOUR_FRAUD_REPO_LINK`
- `YOUR_POWERBI_DASHBOARD_LINK`
- `YOUR_MIRA_REPO_LINK`
- `YOUR_MIRA_DEMO_LINK`

## How to host on GitHub Pages

1. Open your GitHub repository: `DonaRose.github.io`
2. Upload `index.html`, `style.css`, and the `assets` folder.
3. Commit changes.
4. Go to Settings ‚Üí Pages.
5. Select branch: `main`
6. Select folder: `/root`
7. Save.

Your portfolio will be live on your GitHub Pages URL.
