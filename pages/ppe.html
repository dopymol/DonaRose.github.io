PPE Compliance Detection using YOLO

A computer vision system that detects workers and checks whether required safety equipment is present from uploaded images and videos.

View GitHub Contact Me

Problem

Workplace safety monitoring requires checking whether workers are wearing mandatory protective equipment such as helmets, vests, and boots.

Solution

Built a YOLO-based computer vision application that detects workers and PPE items, then identifies compliance status and missing equipment.

Key Features

  • Worker and PPE detection
  • Image and video inference
  • Bounding box visualization
  • Violation identification
  • Streamlit interface for testing

Tech Stack

Python • YOLO • OpenCV • Streamlit • Ultralytics

Project Workflow

Image / Video Upload YOLO Detection Bounding Boxes PPE Matching Violation Check Compliance Output
