MIRA AI Health Predictor

A Django-based healthcare application for managing patient records, generating AI-assisted health remarks, and visualizing health analytics.

View GitHub Contact Me

Problem

Healthcare data needs organized record management, quick patient insights, and simple analytics to support better decision-making.

Solution

Built a Django web application that stores patient reports, validates inputs, generates AI-assisted remarks using Gemini API, and displays analytics dashboards.

Key Features

  • Patient record CRUD operations
  • Gemini API integration for AI remarks
  • Input validation for reports and patient details
  • Dashboard analytics using Chart.js
  • SQLite database integration

Tech Stack

Python • Django • Gemini API • SQLite • Bootstrap • Chart.js
