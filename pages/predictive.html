Predictive Maintenance System

A machine learning dashboard that predicts engine failure risk using NASA turbofan sensor data and explains predictions with SHAP.

View GitHub Contact Me

Problem

Industrial equipment failures can cause downtime and maintenance costs. The goal was to predict failure risk early using sensor readings and engine operating cycles.

Solution

Built an XGBoost-based predictive maintenance model using NASA CMAPSS turbofan data, then integrated SHAP explanations and a Streamlit monitoring dashboard.

Key Features

  • Sensor data preprocessing and feature engineering
  • Failure risk classification
  • Approximately 96% classification accuracy
  • SHAP-based explainability
  • Interactive Streamlit dashboard

Tech Stack

Python • Pandas • Scikit-learn • XGBoost • SHAP • Streamlit
