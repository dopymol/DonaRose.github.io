Credit Card Fraud Detection

A machine learning project for detecting fraudulent financial transactions using imbalanced classification techniques and dashboard-based fraud analytics.

View GitHub Contact Me

Problem

Fraud detection datasets are highly imbalanced, where fraudulent transactions are rare compared to genuine transactions. The challenge was to detect fraud while reducing false alarms.

Solution

Built machine learning models using transaction data, handled imbalance using SMOTE, evaluated fraud-focused metrics, and created Power BI dashboards for business insights.

Key Results

  • Processed 6.3M+ financial transaction records
  • Applied SMOTE for class imbalance handling
  • Achieved 100% fraud precision
  • Achieved 76% fraud recall using Random Forest
  • Built Power BI dashboard for fraud analysis

Tech Stack

Python • Pandas • Scikit-learn • SMOTE • Random Forest • Power BI
